    Detalhes do serviço

    Conheça o Painel PTD!

    Acompanhe o Plano de Transformação Digital da Codevasf — levantamento de serviços e controle de projetos em um só lugar. Vamos fazer um tour rápido?

    Duas abas, dois olhares

    Análise do Levantamento: visão agregada das respostas ao formulário — maturidade digital, integrações, demandas de automação.

    Acompanhamento de Projetos: status individual de cada serviço em andamento no fluxo do PTD.

    A jornada de um serviço

    Cada serviço passa pelas etapas Aguardando área → Triagem UNP → Em TI → Concluído. Essa jornada aparece ao abrir o detalhe de qualquer serviço na tabela.

    Filtros e pesquisa

    Na aba Acompanhamento, use a barra lateral para filtrar por status, risco, área ou prazo. Dentro de cada tabela, o ícone de lupa na barra de título pesquisa rapidamente nos resultados já filtrados.

    Exportar dados

    Marque o checkbox das linhas desejadas (ou "Selecionar todas as linhas", no cabeçalho) e use o botão "Exportar CSV" que aparece na barra acima da tabela.

    Pronto!

    É isso — agora você já conhece o essencial do painel.

    Dúvidas ou sugestões? Fale com a UNP: ae.gpe.unp@codevasf.gov.br ou (61) 2028-4441.

    Declaração de Acessibilidade

    O que foi feito

    Este painel foi construído com atenção contínua à acessibilidade: estrutura semântica (marcos de página, hierarquia de títulos, agrupamento de campos com fieldset/legend), navegação completa por teclado em todos os componentes interativos (abas, menu, modais, carrossel, tooltips, tabela), foco visível em todos os elementos, alternativas em texto para os gráficos, alto contraste, integração com VLibras e áreas de toque com no mínimo 24px. As cores nunca são o único veículo de uma informação — sempre acompanhadas de texto ou ícone.

    Diretrizes seguidas

    WCAG 2.1, nível AA (com alguns critérios de nível AAA atendidos), e o Padrão Digital de Governo (gov.br Design System), versão 3.7.0.

    Limitações conhecidas

    • Os gráficos dependem de JavaScript para renderizar; a descrição textual cobre o essencial dos dados, mas não substitui a exploração detalhada — para isso, use a exportação em CSV das tabelas.
    • O painel depende de conexão com a internet para carregar os dados das planilhas de origem; não há funcionamento offline.
    • Esta declaração foi elaborada com apoio de revisão automatizada e manual; não substitui uma avaliação especializada presencial nem testes com pessoas com deficiência, que ainda não foram realizados para este painel.

    Como reportar um problema

    Encontrou alguma barreira de acessibilidade? Fale com a UNP: ae.gpe.unp@codevasf.gov.br ou (61) 2028-4441.

    Este painel carrega recursos de terceiros para funcionar: fontes (Google Fonts), ícones (Font Awesome via Cloudflare), biblioteca de gráficos (Chart.js via jsDelivr) e os dados do PTD (Google Sheets API). Esses serviços podem registrar informações técnicas de acesso (como IP) segundo suas próprias políticas. O painel em si não coleta nem armazena dados pessoais do usuário.
    • fonts.googleapis.com — fontes Noto Sans/Noto Sans Mono.
    • cdnjs.cloudflare.com — ícones Font Awesome.
    • cdn.jsdelivr.net — biblioteca de gráficos Chart.js.
    • sheets.googleapis.com / docs.google.com — leitura dos dados públicos do Levantamento e Acompanhamento do PTD.

    Configurações avançadas de cookies

    Este painel é uma ferramenta interna de gestão e não define cookies próprios nem usa cookies de rastreamento ou publicidade de terceiros. Por isso não há, abaixo, nenhuma categoria opcional para desativar — só uma explicação transparente do que fica guardado no seu navegador e de quais recursos externos a página carrega para funcionar.

    Armazenamento local essencial
    Ativo

    Guarda no seu navegador (via localStorage, nunca enviado a nenhum servidor) só três preferências: se você já viu este aviso, se já viu o tutorial inicial, e se o modo Alto Contraste está ativado. Nenhum dado pessoal ou identificador de rastreamento é criado. Não pode ser desativado porque a própria função dele é lembrar essas escolhas — sem ele, o aviso e o tutorial apareceriam de novo a cada visita.

    Recursos de terceiros
    Ativo

    A página carrega fontes (Google Fonts), ícones (Font Awesome via Cloudflare), a biblioteca de gráficos (Chart.js via jsDelivr) e os dados do PTD (Google Sheets API). Esses serviços podem registrar informações técnicas de acesso (como IP) segundo suas próprias políticas. Não é possível desativá-los individualmente sem comprometer o funcionamento do painel — ícones, gráficos e dados deixariam de aparecer.

    Reportar um erro

    Descreva o que aconteceu. O painel já tenta capturar um print da tela automaticamente para anexar — seu navegador vai pedir para você escolher a tela/aba a compartilhar (é a única forma que os navegadores permitem; não existe captura silenciosa).
